# Tagging Service Documentation

This section contains the documentation for the Tagging Service.

The Tagging Service provides a flexible, hierarchical tagging system designed to support structured metadata for entities (e.g., images, film stills, documents). It powers a purpose-aware Inspector User Interface (UI) and supports dynamic metadata rendering, tagging logic, and rich relationships between tags.
